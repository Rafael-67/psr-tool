# PSR Tool v2.0-compat

Compatibility release based directly on PSR Tool v1.4.

## Purpose

This release preserves the complete v1.4 functionality while aligning the public-facing scientific reference with the final published article and its Supplementary Appendix S1.

## Preserved functionality

- Bilingual EN/ES guided 8-step PSR assessment.
- Nine procedural modulating factors.
- ≥2 unfavorable factors escalation rule.
- Likelihood × severity integration.
- BSL assignment and control recommendations.
- Structured downloadable reports.
- Persistent institutional platform and reviewer profiles.
- Built-in worked examples.
- CSV export and audit data.
- Cohen's kappa and Fleiss' kappa analytics.
- Existing `psr:*` localStorage compatibility.

## Scientific reference

Cena-Diez R. An 8-step procedure-specific risk framework enables reproducible biosafety level assignment beyond agent-based classification. Front Bioeng Biotechnol. 2026;14:1879247. doi:10.3389/fbioe.2026.1879247.

## Scope of this release

No scientific calculation rules have been replaced. No storage key has been renamed. No module has been removed. This is the controlled baseline for subsequent v2 development.
